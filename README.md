# CodeAlpha Hangman Game

A simple and interactive Hangman word-guessing game developed using Python as part of the CodeAlpha internship project.

## Features

* Randomly selects a word from a predefined list.
* Allows the player to guess one letter at a time.
* Displays the current progress of the word.
* Allows a maximum of 6 incorrect guesses.
* Prevents repeated letter guesses.
* Validates user input.
* Displays winning and game-over messages.

## Technologies Used

* Python
* Random Module
* Set Data Structure

## How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code or a terminal.
4. Run the following command:

```bash
python hangman.py
```

## Game Output

The game displays the hidden word using underscores and updates the word as the player guesses the correct letters.

Example:

```text
Word: _ _ t _ _ n
Incorrect guesses remaining: 5
Guessed letters: a, n, t

Guess a letter:
```
